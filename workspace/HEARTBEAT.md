# Heartbeat Checklist

## Daily Checks (2-4x per day)

### Morning Check (08:00)
- [ ] Check calendar for today's events
- [ ] Check email for urgent messages
- [ ] Check notifications/mentions
- [ ] Weather check (if relevant)

### Afternoon Check (12:00)
- [ ] Calendar reminders (<2h upcoming events)
- [ ] Quick inbox scan
- [ ] Any pending tasks?

### Evening Check (18:00)
- [ ] Tomorrow's calendar preview
- [ ] Follow up on pending items
- [ ] Summary of day's activities

## Memory Maintenance (Weekly)
- [ ] Review week's memory files
- [ ] Update MEMORY.md with key learnings
- [ ] Clean up outdated information

## Rules

### When to Reply
- HEARTBEAT_OK if nothing urgent
- Reach out if:
  - Email urgent
  - Event <2h away
  - >8h since last interaction

### When to Stay Quiet
- Late night (23:00-08:00)
- Just checked <30 min ago
- Nothing new
- Human clearly busy
