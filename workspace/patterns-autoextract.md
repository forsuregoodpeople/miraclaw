# Auto-Extract Patterns

Patterns for automatic preference extraction from user messages.

## Name Patterns

Prefixes:
- panggil saya
- panggil aku
- call me
- panggil gue

Label: user wants to be called

## Name Declaration

Prefixes:
- nama saya
- nama ku
- namaku
- my name is

Label: user's name is

## Like Patterns

Prefixes:
- saya suka
- aku suka
- gue suka
- i like
- i love

Label: user likes

## Dislike Patterns

Prefixes:
- saya tidak suka
- aku tidak suka
- i don't like
- i dislike

Label: user dislikes

## Location Patterns

Prefixes:
- saya tinggal di
- aku tinggal di
- saya di
- i live in

Label: user lives in

## Work Patterns

Prefixes:
- saya kerja di
- aku kerja di
- saya bekerja di
- i work at

Label: user works at
